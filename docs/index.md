**Doritus** is an address book software for NUS teaching staff to manage student contacts. While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using Doritus, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested in developing Doritus, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
